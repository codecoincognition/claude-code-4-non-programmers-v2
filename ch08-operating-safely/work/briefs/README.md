# work/briefs/

This folder is where the Chapter 7 morning brief writes its dated brief files
(e.g. `2026-05-06.md`). The `show-brief.sh` SessionStart hook reads the most
recent `*.md` here and prints it when a new Claude session starts.

It's also the folder at the center of this chapter's cold open — the one Maya
"cleaned up" with a one-line prompt that deleted 47 files instead of 9. The
whole leash you build in Chapter 8 exists so that the next "clean up the
drafts" prompt pauses at a plan you can read before anything is deleted.

A tiny synthetic sample brief (`2026-05-06.md`) sits alongside this README so
`show-brief.sh` has something to display when you try the setup. Replace it with
your own brief files (or let Chapter 7's morning-brief command generate them).
