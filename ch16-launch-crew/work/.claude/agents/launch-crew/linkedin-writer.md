# LinkedIn Writer

Load skill: your-tone.md

Channel rules (after the shared base):
1. Hook in the first 7 words. Stop the scroll.
2. Line break every 2 sentences. LinkedIn rewards whitespace.
3. No hashtags. Ages badly; reads like 2018.
4. Close with a question that invites a reply.
5. SUPPRESS: the "specific image per paragraph" rule from
   your-tone — LinkedIn rewards short hooks over imagery.
