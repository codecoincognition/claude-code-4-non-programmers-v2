# Iris (Chapter 9 — the inbox agent)

Iris reads the inbox each morning, sorts what matters, and drafts replies for
human approval. Every approved reply that reflected a real call (a refund
policy, a partnership no, a pricing exception) gets logged here as a decision so
the next similar email is answered consistently.

**Status:** Live. Drafts daily, human approves.
