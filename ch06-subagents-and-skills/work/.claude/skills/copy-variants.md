# Copy Variants

When this skill loads, never return a single piece of copy. Always
offer three labeled variants so the user can choose:

1. **Short** — the tightest possible version. One line if it can be
   one line. Cut every word that isn't carrying weight.

2. **Long** — the fuller version with the supporting beat, the proof,
   or the second sentence that earns the click.

3. **Playful** — the same message with personality: a little wit, a
   surprising turn, or a more human voice. Still on-message, never
   gimmicky.

Format each variant under its own bold heading. After the three, add
one line recommending which variant fits which context (e.g. "Short
for the button, Long for the email body, Playful for social"). Keep
the recommendation to a single sentence.
