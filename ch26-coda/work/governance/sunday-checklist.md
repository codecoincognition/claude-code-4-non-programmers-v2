# Sunday-Night Governance Checklist

1. Walk ~/work/.claude/ — every agent, every command, every skill, every hook. Note anything last invoked >4 weeks ago.
2. Read ~/work/watchdog/escalations.log for the past week — identify any false-positive pattern worth tightening.
3. For each agent: invoked in 4-8 weeks? Superseded by something newer? Decision: rebuild, retire, or keep.
4. Update CLAUDE.md if any standing instruction has drifted.
5. Move retired agents to .claude/retired/ — don't delete; they're institutional memory.
