# Governance log

Append-only. One line per fire / keep / rebuild decision. Date, what you did,
one-clause reason. Newest entries at the top.

The point is not the prose. The point is that six months from now, when you
wonder *"what happened to research-helper,"* this file answers in one breath.

Format:

```
YYYY-MM-DD: {verb} {artifact-name} — {one-clause reason}.
```

Verbs: `retired`, `rebuilt`, `kept`, `restored`. Use `restored` when you pull
an agent back out of `~/work/.claude/retired/` because the need came back.
