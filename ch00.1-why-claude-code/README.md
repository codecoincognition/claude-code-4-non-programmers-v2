# Chapter 0.1 — Why Claude Code

This chapter is positioning, not practice. It teaches two things:

1. **The wedge sentence** — a one-line way to describe what Claude Code is to a non-programmer friend without lying or oversimplifying.
2. **The agentic filter** — a mental test for deciding when reaching for an agent is the right move (and when it isn't).

## What you'll find in this folder

Nothing clone-able. There are no scripts, configs, or worked examples here because the chapter is orienting prose — it sets up the rest of the book.

The clone-able artifacts start arriving in Chapter 0.2 (your first install and prompts) and Chapter 0.3 (your first `settings.local.json`). From Chapter 1 onward, every chapter ships a folder you can copy into `~/work/`.
