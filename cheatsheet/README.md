# Power-User Command Cheat Sheet

A printable, one-page lookup for every power-user command in *Claude Code for Nonprogrammers* (chapters 22-24). This is the companion-repo copy of **Appendix I**.

## What's here

- **`commands.md`** — the full cheat sheet: a quick alphabetical lookup table, a by-family table, common compositions, and a detail entry for each command.

## How to use it

- **Looking a command up?** Start with the alphabetical table in `commands.md`, then scroll to the matching entry under "Entry detail."
- **Not sure which command you want?** Use the "By family" table to find the right neighborhood (Navigation, Awareness, Review, etc.), then drill in.
- **Want to print it?** `commands.md` is structured to print cleanly. Render to PDF with your tool of choice (or `/make-pdf` if you have it).

## Version note

This reflects the Claude Code v2.x command surface as of the book's draft date. The surface evolves — commands may be renamed or restructured. The errata page is the live truth:

https://github.com/codecoincognition/claude-code-for-nonprogrammers-v2/blob/main/ERRATA.md

If an entry here doesn't match what your terminal does, the errata page wins.
